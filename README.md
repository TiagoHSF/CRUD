# CRUD
CRUD Respository | TYPESCRIPT, SEQUELIZE e MYSQL

  Using in Project
   - Plugins ESLINT & PRETTIER
   - GIT
      - .gitignore
   - NPM
      - npm init
   - .editorconfig
      - ident size 2
      - crlf to lf
   - Typescript
      - npm add -D typescript
      - tsc -init
         - tsconfig.json
              - "target": "es2017", "allowJs": true, "outDir": "./dist"
   - Eslint
     - npm add -d eslint
     - npm install eslint-init -g
     - eslint-init --node
   - Dotenv
   - Scripts desenvolvimento, build, start
